Responsive Login Page - Glassmorphism Style 🌌
Este projeto consiste em uma tela de login moderna e responsiva, desenvolvida para praticar técnicas avançadas de CSS3, com foco em estética visual e experiência do usuário (UX).

🚀 Tecnologias Utilizadas
HTML5: Estruturação semântica do formulário de acesso.

CSS3: * Glassmorphism: Utilização de backdrop-filter: blur e transparências para criar o efeito de vidro.

Flexbox: Alinhamento centralizado e preciso dos elementos na tela.

Responsividade: Ajuste dinâmico para garantir uma boa visualização em diferentes tamanhos de dispositivos.

Google Fonts: Integração de fontes personalizadas para melhor legibilidade.

🛠️ Funcionalidades e Destaques
Design Moderno: Interface limpa com fundo imersivo e elementos translúcidos.

Interatividade nos Inputs: Campos de formulário com ícones integrados (Usuário e Senha).

Feedback Visual: Botão de login estilizado com efeitos de hover e transições suaves.

Links de Recuperação: Inclusão de "Esqueci minha senha" e opção de cadastro para novos usuários.

🧠 Aprendizados
Durante o desenvolvimento deste projeto, pude consolidar conhecimentos sobre o posicionamento de elementos com Flexbox e a aplicação de filtros de desfoque no fundo para destacar o conteúdo principal sem perder a conexão visual com a imagem de fundo.
